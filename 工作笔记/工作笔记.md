### 日常工作笔记



#### JavaScript微信号正则校验

1、可以使用6-20个子母、数字、下划线和减号
2、必须以字母开头（字母不区分大小写）
3、不能设置中文

```js
/^[a-zA-Z]([-_a-zA-Z0-9]{5,19})$/
```

#### 小程序调试嵌套的H5页面

#### 默认进入指定页面

在page.json文件中修改

```js
  "condition" : { //模式配置，仅开发期间生效
    "current": 0, //当前激活的模式(list 的索引项)
    "list": [
      {
        "name": "name", //模式名称 name随便写
        "path": "pages/user/retake", //启动页面，必选
        "query": "" //启动参数，在页面的onLoad函数里面得到
      }
    ]
  },
```
